## 🎯 프로젝트 핵심 가치

**'나의 잠재력'과 '세상의 요구'를 연결하는 내비게이션**

단순한 직업 추천을 넘어, AI 기반의 **'세상 흐름 분석 엔진'**을 구축하여 청년 세대에게 주도적으로 삶을 설계할 수 있는 객관적인 통찰력을 제공합니다.

## 📊 주요 기능

1. **AI 트렌드 대시보드**
   - 트렌드 속도계
   - 세상의 요구 지도
   - 경제 상황 요약

2. **개인화 기능**
   - 개인 맞춤형 필터링
   - 트렌드 기반 역량 로드맵
   - AI 트렌드 대화 코치
   - 직무 키워드 브릿지

## 🔄 데이터 전략

- **선행 지표 중심**: 후행 지표가 아닌 사람들의 행동 데이터 분석
- **3대 핵심 지표**: 돈의 흐름, 혁신의 흐름, 역량의 흐름
- **피드백 루프**: 사용자 행동 데이터를 통한 지속적인 개인화 개선
