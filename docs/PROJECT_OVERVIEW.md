# 🚀 AI 기반 청년 글로벌 통찰력 서비스 기획안 (MVP)

## 1. 프로젝트 개요 및 배경

### 1.1. 프로젝트 목표 (Goal)

AI 기반의 **'세상 흐름 분석 엔진'**을 구축하여, 진로에 대한 막연함과 불안을 느끼는 청년 세대에게 **주도적으로 삶을 설계할 수 있는 객관적인 통찰력**을 제공합니다.

### 1.2. 프로젝트 핵심 가치 (Core Value)

단순한 직업 추천을 넘어, **'나의 잠재력'과 '세상의 요구'를 연결하는 내비게이션** 역할 수행.

### 1.3. 대상 (Target)

자신이 뭘 해야 할지 모르는 10대 후반 ~ 30대 초반의 청년 및 예비 사회인.

---

## 2. 서비스 핵심 기능 구성 (MVP)

본 서비스의 핵심은 **'AI 트렌드 대시보드'**이며, 모든 서브 기능은 이 대시보드의 통찰력을 활용합니다.

### 2.1. 메인 기능: AI 트렌드 대시보드 (Global Insight Dashboard)

|**기능명**|**내용 및 역할**|**활용 데이터**|
|---|---|---|
|**트렌드 속도계**|현재 가장 빠르게 성장하거나 주목받는 **핵심 트렌드 5가지**의 속도(🔥, 🚀 등)를 시각화하여 제시.|뉴스/블로그 언급량 변화, Google/Naver **검색량 변화율**, 스타트업 투자 동향.|
|**세상의 요구 지도**|시장이 해결하지 못하고 있어 **기회(Opportunity)**가 되는 분야(경제적/사회적 결핍)를 도출하여 제시.|**Crowdfunding 인기 분야**, WEF 등 글로벌 기관의 **미래 과제 보고서** 키워드 분석.|
|**경제 상황 요약**|복잡한 금리, 환율, 주요 지수 변동을 청년의 언어로 **쉽게 해석**하고 삶에 미치는 영향을 요약 제시.|공공 금융 API (금리, 환율), 주요 경제 언론사 기사 요약.|

### 2.2. 서브 기능 (트렌드 활용 방안)

| **기능명**           | **내용 및 역할**                                                             | **연관성**                  |
| ----------------- | ----------------------------------------------------------------------- | ------------------------ |
| **개인 맞춤형 필터링**    | 청년 개인의 성향/가치관 데이터를 분석하여, 메인 트렌드 중 **가장 관련성이 높은 3가지**를 하이라이트.            | **나의 길 제시** (핵심 논리)      |
| **트렌드 기반 역량 로드맵** | 관심 트렌드를 선택하면, 필요한 **핵심 역량 3가지**와 그 역량을 기를 수 있는 **온라인 강의/도서** 목록을 연결 제시. | **실행력** (What to do)     |
| **AI 트렌드 대화 코치**  | 사용자의 트렌드 관련 질문에 대해 **수집된 데이터를 근거**로 답변하고, 개인의 성향에 맞는 해석을 추가 제공.         | **능동적 학습** (Q&A)         |
| **직무 키워드 브릿지**    | 트렌드를 활용하는 **미래 유망 직무 3가지**의 핵심 키워드를 제시하여, 추가 탐색(현직자 검색 등)을 유도.          | **진로 탐색** (Sub-function) |

## 3. 데이터 수집 및 분석 전략

### 3.1. 데이터 소스 및 난이도 관리 (4개월 MVP)

| **데이터 유형**  | **주요 수집처**                                            | **난이도** | **수집/분석 전략**                                     |
| ----------- | ----------------------------------------------------- | ------- | ------------------------------------------------ |
| **트렌드 텍스트** | 네이버/구글 뉴스 API, WEF/컨설팅 펌 RSS, 기술 전문 매체 RSS            | 중하      | API/RSS 위주 수집, **GPT API 활용 요약 및 분류** (전처리 단순화). |
| **행동 데이터**  | Google Trends API, 국내외 베스트셀러 API, Crowdfunding 랭킹 페이지 | 하       | **검색량/판매량 등 수치 데이터** 활용하여 트렌드 속도계 지표로 사용.        |
| **개인 데이터**  | 서비스 내 심리/가치관 **설문 모듈**                                | 하       | 사용자가 직접 입력한 데이터를 DB에 구조화하여 저장.                   |

### 3.2. AI 분석의 핵심 (GPT-API 활용)

- **텍스트 처리:** 수집된 텍스트의 **요약, 키워드 추출, 감성/정서 분석** 및 **트렌드 카테고리 분류**를 GPT API에 위임하여 개발 시간 단축 및 분석 정확도 확보.
    
- **매칭 로직:** 사용자 데이터와 트렌드 역량 데이터를 **규칙 기반(Rule-based)**으로 매칭하는 초기 로직 구현 후, 향후 사용 데이터를 기반으로 추천 알고리즘 고도화.

## 4. 개발 계획 (4개월 1인 개발 로드맵)

|**기간**|**주요 목표**|**핵심 구현 기능**|
|---|---|---|
|**1개월 차**|**기반 구축**|웹/앱 기본 구조, 사용자 DB, **핵심 데이터 수집 파이프라인 (API/RSS)** 1개 구축.|
|**2개월 차**|**분석 로직**|**GPT-API 연동** (트렌드 요약/분류), **트렌드 속도계** (변화율 계산) 초기 로직 구현.|
|**3개월 차**|**대시보드 완성**|**메인 대시보드 시각화** (차트, 게이지), **개인 필터링** 및 **직무 키워드 연결** 기능 구현.|
|**4개월 차**|**안정화 및 배포**|서브 기능 추가(AI 코치 초기 버전), 버그 수정, UX/UI 개선 및 MVP 서비스 런칭.|

## 5. 기대 효과

- **청년:** 복잡한 세상 속에서 객관적인 위치를 파악하고, 자신의 강점과 연결된 **구체적인 행동 지침**을 얻어 막연한 불안감 해소.
    
- **서비스:** 단순 정보 제공을 넘어, **데이터 기반의 통찰력**을 제공하는 차별화된 **AI 멘토링 플랫폼**으로 포지셔닝.
