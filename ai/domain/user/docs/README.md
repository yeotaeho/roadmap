## 🎯 User Domain 개요

User Domain은 사용자의 역량, 로드맵, 그리고 AI 에이전트와의 상호작용을 관리하는 도메인입니다.

### 주요 기능

1. **사용자 역량 관리**
   - 스킬 및 숙련도 추적
   - 역량 갭(Gap) 분석

2. **로드맵 관리**
   - 학습 진척도 추적
   - 목표 트렌드 연결

3. **AI 에이전트 상호작용**
   - LangGraph 기반 대화형 AI 코치
   - 개인화된 인사이트 생성

## 🔗 관련 도메인

- **OAuth Domain**: 사용자 인증 및 기본 프로필 관리
- **News Domain**: 트렌드 데이터 수집 및 분석
