# Project Context: Global Pulse Dashboard (청년 인사이트)

## 1. 프로젝트 개요 (Overview)
- **목적:** 청년들에게 "세상의 흐름(트렌드)"과 "나의 기회"를 연결해주는 진로/시장 분석 플랫폼.
- **철학:** 단순한 뉴스 나열이 아닌, 데이터 기반의 모멘텀(가속도)과 인과관계를 직관적으로 시각화하여 사용자가 '행동'할 수 있는 확신을 제공.
- **핵심 UI 패턴:** 블룸버그 터미널(Market Terminal)의 밀도 있는 데이터 배치와 벤토 박스(Bento Box) 스타일의 깔끔한 카드 UI를 혼합.

## 2. '실시간 펄스(Pulse)' 탭 UI/UX 명세서
대시보드의 첫 번째 L2 탭으로, 세상의 거시적 변화를 한눈에 조망하는 섹션.

### 2.1. 컬러 시스템 (Youth & Tech)
- **Primary:** Vibrant Indigo (`#4F46E5`), Blue (`#2563EB`) - 전문성, AI, 신뢰
- **Secondary (Point):** Mint/Emerald (`#10B981`), Purple (`#8B5CF6`) - 상승, 기회, 통찰

### 2.2. 레이아웃 구조 (Top-Down 하이브리드 Bento)
1. **글로벌 펄스 헤더(최상단):**
   - 현재 속도(예: 180km/h), 전일 대비 증감(%), 주간 지수(0~100), 한 줄 인사이트를 하나의 메인 카드로 표시.
2. **관계 신호 영역(중단):**
   - **Top 섹터 히트맵/매트릭스:** 분야 x 상태를 격자로 표현하고, 색 강도(속도) + 아이콘(기회/리스크)으로 빠르게 스캔.
   - **인과관계 체인:** `[거시 이벤트] -> [산업 영향] -> [청년 기회]` 3스텝 플로우 카드.
3. **트렌드 증명 영역(중단/하단):**
   - **세대교체/크로스오버 라인 차트:** 기존 수요 하락선 vs 신흥 수요 상승선의 교차 시점 강조.
   - **연간 글로벌 모멘텀 (Line/Area Chart):** 중장기 흐름 확인.
4. **보조 요약 영역(하단):**
   - **3줄 경제 브리핑:** 거시 지표를 청년 관점으로 해석.
   - **실시간 키워드 티커:** 라이브 감각 강화.
   - **키워드 클라우드/점유율:** 보조 맥락 정보 제공.

### 2.3. 시각화 해석 규칙
- 펄스 탭은 필터 탐색이 아닌, "한 화면 조망"을 우선한다.
- 사용자 시선 흐름: 요약(헤더) -> 관계/전환(매트릭스/체인/크로스오버) -> 라이브 피드(브리핑/티커).
- 각 카드에는 수치와 함께 "한 줄 결론"을 포함한다.

## 3. 백엔드 엔진 및 데이터 파이프라인 (예정)
- 프론트엔드를 먼저 구축한 후, 트렌드 데이터를 수집·가공하는 **'룰 기반(Rule-based) 계산 엔진'**을 백엔드에 연동할 예정.
- 복잡한 생성형 AI 추론보다는, 특정 키워드의 검색량, 투자액, 관련 공고 수 등을 수집하여 명확한 로직(가중치 합산 등)을 통해 점수(0~100)와 속도(km/h)를 도출함.

## 4. LLM 작업 지침서 (Instruction for AI)
- 이 문서를 읽은 후, 컴포넌트나 로직을 작성할 때는 항상 **Next.js (App Router), Tailwind CSS** 환경을 전제로 코드를 생성할 것.
- 차트 구현이 필요할 경우 `recharts` 라이브러리 사용을 우선적으로 고려할 것.
